#功能说明
+ 打印文字
+ 打印对齐
+ 打印二维码

#问题排查
- 确定手机蓝牙打开,允许微信访问蓝牙的权限
- 微信要允许访问蓝牙和位置信息
- 重启蓝牙设备(打印机)
- 不行的时候,删掉手机中蓝牙设备(打印机),重新链接
